# ADR-0002 Tech Stack Standard

## Status
Accepted

## Chosen Stack
- Frontend: Next.js + React + TypeScript + Tailwind
- Backend: Next.js Route Handlers
- ORM/DB: Prisma + SQLite (MVP), PostgreSQL target
- Auth: NextAuth + Google OAuth
- External APIs: YouTube Data API v3, Google Calendar API
- Testing: Playwright (E2E), ESLint

## Why
- Tek repo icinde hizli gelistirme
- Type-safe model ve migration kolayligi
- OAuth ve API entegrasyonlarina hizli uyum
